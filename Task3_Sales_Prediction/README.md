# 📊 Sales Prediction using Machine Learning

## 📌 Project Overview

This project aims to predict product sales based on advertising expenditure across different media channels such as TV, Radio, and Newspaper.
A Linear Regression model is used to analyze the relationship between advertising budgets and sales performance.

---

## 🎯 Objectives

* Understand the impact of advertising on sales
* Perform data analysis and visualization
* Build a predictive model using machine learning
* Evaluate model performance

---

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

---

## 📂 Dataset

The dataset used in this project contains the following features:

* **TV**: Advertising budget spent on TV
* **Radio**: Advertising budget spent on Radio
* **Newspaper**: Advertising budget spent on Newspaper
* **Sales**: Product sales

---

## ⚙️ Project Workflow

1. Data Collection
2. Data Preprocessing
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Model Building (Linear Regression)
6. Model Evaluation
7. Prediction

---

## 📊 Exploratory Data Analysis

* Pairplot used to visualize relationships between variables
* Correlation heatmap used to identify feature importance
* Strong correlation observed between TV advertising and Sales

---

## 🤖 Model Used

* **Linear Regression**

The model predicts sales using the formula:

Sales = a(TV) + b(Radio) + c(Newspaper) + constant

---

## 📈 Model Evaluation

The model was evaluated using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

👉 The model achieved a high R² score (~0.9), indicating good prediction accuracy.

---

## 🔍 Key Insights

* TV advertising has the strongest impact on sales
* Radio advertising has moderate impact
* Newspaper advertising has the least impact

---

## 🚀 Example Prediction

Input:

* TV = 200
* Radio = 40
* Newspaper = 50

Output:

* Predicted Sales ≈ 20+ units

---

## 📌 Conclusion

The Linear Regression model successfully predicts sales based on advertising data.
This project demonstrates how businesses can use data-driven strategies to optimize marketing budgets.
