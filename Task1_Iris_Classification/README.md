# 🌸 Iris Flower Classification

This project uses machine learning to classify Iris flowers based on sepal and petal measurements.

## 📊 Dataset
- Iris dataset (sepal length, sepal width, petal length, petal width)

## 🤖 Model Used
- Decision Tree Classifier

## 📈 Accuracy
- ~96%

## 🛠 Tools & Libraries
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run
1. Install required libraries
2. Run:
   python iris.py

## 📌 Conclusion
The model successfully classifies Iris flower species with high accuracy.
