# 📊 Unemployment Analysis

This project focuses on analyzing unemployment trends using Python.

## 📌 Objective
The goal of this project is to understand how unemployment rates change over time and across different states.

## 📂 Dataset
- State-wise unemployment data

## 🛠 Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📈 Analysis Performed
- Comparison of unemployment rates between states
- Visualization of trends over time
- Basic insights from the data

## ▶️ How to Run
1. Install required libraries
2. Run the Python file:
   python unemployment.py

## 📌 Conclusion
This project helps in understanding unemployment patterns using simple data analysis and visualization techniques.
